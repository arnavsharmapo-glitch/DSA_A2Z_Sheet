# Daily DSA Workflow — Quick Reference

## The Loop (every time you solve a problem)

1. **Solve** the problem on TUF / LeetCode / NeetCode
2. **Save** it in VS Code, inside the right topic folder in `DSA_A2Z_Sheet`
3. **Add the header comment:**
```cpp
   // Problem N: <Problem Name>
   // Link: <problem URL>
   // Difficulty: Easy / Medium / Hard
   // Approach: <one-line note on your method>
```
4. **Commit + push** (see commands below)
